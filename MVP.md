# 图片背景移除工具 - MVP 需求文档

> 版本：v1.0  
> 日期：2026-03-26  
> 状态：初稿

---

## 1. 项目概述

| 项目名 | SnapBackground |
|--------|---------------|
| 一句话描述 | 帮助用户快速移除图片背景的在线工具 |
| 核心价值 | 简单、快速、免费，无需下载安装 |
| 目标用户 | 需要处理产品图、证件照、电商素材的个人/小商家 |

---

## 2. 技术架构

### 2.1 技术栈

```
前端：     Next.js 15 (App Router)
部署平台： Cloudflare Pages
后端：     Cloudflare Workers (Node.js Runtime)
AI API：   Remove.bg (https://remove.bg/api)
域名：     Workers 直接代理，无独立后端服务器
```

### 2.2 数据流

```
用户浏览器
    ↓ (HTTPS multipart/form-data)
Cloudflare Worker
    ↓ (HTTPS fetch)
Remove.bg API
    ↓
Cloudflare Worker
    ↓ (流式返回)
用户浏览器 → 下载
```

**关键：无持久化存储，图片不落磁盘**

### 2.3 限制

| 限制项 | 值 | 说明 |
|--------|-----|------|
| 单文件最大 | 12MB | Remove.bg 免费版限制 |
| Workers 请求体 | 25MB | Cloudflare 免费版 |
| API 免费额度 | 50次/小时 | Remove.bg 免费版 |
| 并发 | 1次/用户 | 防止滥用 |

## 3. 功能需求

### 3.1 MVP 功能列表

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| F1 | 拖拽/点击上传 | P0 | 支持拖拽和点击选择文件 |
| F2 | 图片预览 | P0 | 上传后显示原图预览 |
| F3 | 一键移除背景 | P0 | 点击按钮触发处理 |
| F4 | 结果展示 | P0 | 显示处理后的透明背景图 |
| F5 | 下载结果 | P0 | PNG 格式一键下载 |
| F6 | 进度指示 | P1 | 处理中显示 loading 状态 |
| F7 | 错误提示 | P1 | API 失败时友好提示 |
| F8 | 文件大小提示 | P2 | 超过 12MB 时提前告知 |

### 3.2 用户流程

```
[上传图片] → [预览原图] → [点击移除] → [处理中...] → [下载结果]
                   ↓
              [错误提示]
```

### 3.3 边界情况处理

| 场景 | 处理方式 |
|------|----------|
| 文件 > 12MB | 提示"图片请小于 12MB"，不调用 API |
| 非图片文件 | 仅接受 image/jpeg, image/png, image/webp |
| API 调用失败 | 显示错误信息，提供重试按钮 |
| 网络超时 | 30s 超时提示 |
| 并发请求 | 禁用按钮，防止重复提交 |

## 5. API 设计

### 5.1 Worker 端点

```
POST /api/remove-background
Content-Type: multipart/form-data

请求：
  - image: File (图片文件)

响应（成功）：
  Content-Type: image/png
  状态码: 200
  Body: PNG 二进制流

响应（错误）：
  Content-Type: application/json
  状态码: 4xx/5xx
  Body: { "error": "错误描述" }
```

### 5.2 Remove.bg API 调用

```javascript
const formData = new FormData();
formData.append('image_file', file);
formData.append('size', 'auto');

const response = await fetch('https://api.remove.bg/v1.0/removebg', {
  method: 'POST',
  headers: {
    'X-Api-Key': REMOVE_BG_API_KEY,
  },
  body: formData,
});
```
